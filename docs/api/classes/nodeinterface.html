registerListener

  • registerListener(thisValue: any, cb: ListenerType): void
    • Defined in model/nodeInterface.ts:60

    Register a callback function that is called whenever the value of the interface changes. Used primarily for connections to work (they will "transmit" the value from one interface to another when the value changes.) Note: Inline ES6 arrow functions won't work as you need a reference to the callback later to unregister the listener.

    Parameters

    • thisValue: any

      The value this will be bound to in the callback function.

    • cb: ListenerType

      The callback function that will be called with the new value as parameter.

    Returns void
